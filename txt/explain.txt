- stack data structure (linear data structure)
    A stack is a data structure 
    that follows a last in, first out (LIFO) protocol. 
    The latest node added to a stack is the node 
    which is eligible to be removed first. 
    If three nodes ( a , b and, c ) are added 
    to a stack in this exact same order, 
    the node c must be removed first.
